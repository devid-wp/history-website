# Віртуальний Історичний Музей

Цей проект є веб-додатком на базі Flask, який демонструє археологічні культури України через їхню кераміку та інші артефакти.

## Структура проекту (mkdir style)

```text
history website/
├── app.py              # Основний файл сервера Flask
├── static/             # Статичні файли (CSS, зображення, JS)
│   └── (зображення експонатів)
├── templates/          # HTML шаблони (Jinja2)
│   ├── index.html      # Головна сторінка
│   └── item.html       # Сторінка деталей експоната
├── information/        # Додаткові матеріали та документація
└── venv/               # Віртуальне середовище Python
```

## Вимоги для роботи

Для успішного запуску проекту вам знадобиться:

1.  **Python 3.x**
2.  **Flask** (веб-фреймворк)

## Як налаштувати та запустити

1.  **Створення та активація віртуального середовища:**
    ```bash
    python3 -m venv venv
    source venv/bin/activate  # Для Linux/macOS
    # або
    .\venv\Scripts\activate  # Для Windows
    ```

2.  **Встановлення залежностей:**
    ```bash
    pip install flask
    ```

3.  **Запуск програми:**
    ```bash
    python app.py
    ```

4.  **Перегляд у браузері:**
    Відкрийте [http://127.0.0.1:5002](http://127.0.0.1:5002)

## Основні функції

*   **Головна сторінка:** Відображає список археологічних культур (Трипільська, Ямна, Катакомбна та ін.) з коротким описом та зображенням.
*   **Деталі експоната:** При натисканні на картку відкривається повний опис культури, період її існування та регіон поширення.
*   **Динамічні маршрути:** Використання Flask для відображення контенту на основі ID експоната.
